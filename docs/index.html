Features

Date Range Control

Specify any start and end date to generate commits across your desired timeframe.

Realistic Patterns

Generates 15-20 commits per day with natural variation, skipping Sundays for authenticity.

System Time Management

Automatically handles NTP disabling and system time restoration for seamless operation.

Bash Script

Simple, lightweight bash script that works on Linux and macOS systems.

How It Works

1

Disable NTP

Temporarily disables Network Time Protocol to allow manual system time changes.

2

Iterate Date Range

Loops through each day in your specified date range, intelligently skipping Sundays.

3

Generate Commits

Creates 15-20 commits per day with backdated timestamps and realistic content.

4

Restore & Push

Restores your system date and pushes all commits to your main branch.

Usage

Command Syntax

sudo ./auto-commit.sh <START_DATE> <END_DATE> <REPO_DIR>

Example

sudo ./auto-commit.sh 2023-01-01 2023-12-31 /path/to/your/repo

Requirements

  • Linux or macOS system
  • Git installed and configured
  • Sudo privileges (for system time changes)
  • Existing Git repository

Important Disclaimer

This script is designed for educational and personal fun use only. It should not be used to mislead others about your actual contribution history or programming activity. Use responsibly and be transparent about generated contributions when appropriate.
